Exercices HTML et accessibilité

Choisis un exercice ci-dessous pour t’entraîner au HTML et à l’accessibilité du web.

  • 30 défis HTML QCM pour apprendre le langage HTML (structure, balises, formulaires…). Niveau 3ème.
  • 30 exercices — Écrire du HTML Écris le code HTML demandé dans chaque zone (paragraphes, liens, listes, formulaires…).
  • 30 défis accessibilité QCM pour découvrir l’accessibilité du web (lecteurs d’écran, contrastes, formulaires…).
  • Erreurs d’accessibilité — Boutique en ligne Page contenant des erreurs d’accessibilité à repérer avec un lecteur d’écran (TalkBack, VoiceOver, NVDA).

HTML = structure, CSS = apparence, JavaScript = comportement. L’accessibilité permet à tous d’utiliser le web.
